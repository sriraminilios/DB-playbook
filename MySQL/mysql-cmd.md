# MySQL Reference Documentaion

## List all databases
```
SHOW DATABASES;
```
## Use a Database
```
USE database;
```
## List all Tables
```
SHOW TABLES;
```
## List running process
```
SHOW PROCESS
```
## List running process full SQL text
```
SHOW FULL PROCESS
```

## Show innodb status
```
SHOW ENGINE INNODB STATUS;
```
