# MySQL Reference Documentaion

## Basic Mysql shell commands

### List all databases
```
SHOW DATABASES;
```
### Use a Database
```
USE database;
```
### List all Tables
```
SHOW TABLES;
```
### Describe a Table
```
DESC tablename;
```
### Get table schema
```
SHOW CREATE TABLE tablename;
```
### List running process
```
SHOW PROCESSLIST;
```
### List running process full SQL text
```
SHOW FULL PROCESSLIST;
```
### Show innodb status
```
SHOW ENGINE INNODB STATUS\G
```
