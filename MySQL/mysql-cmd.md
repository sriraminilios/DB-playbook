# MySQL Reference Documentaion

### List all databases
```
SHOW DATABASES;
```
### Use a Database
```
USE database;
```
### List all Tables
```
SHOW TABLES;
```
### List running process
```
SHOW PROCESSLIST;
```
### List running process full SQL text
```
SHOW FULL PROCESSLIST;
```

### Show innodb status
```
SHOW ENGINE INNODB STATUS\G
```
